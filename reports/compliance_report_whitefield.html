Methodology

  1. Data Acquisition: Sentinel-2 L2A imagery via Microsoft Planetary Computer STAC API
  2. Preprocessing: Cloud masking (SCL), atmospheric correction (L2A), band resampling (20m to 10m)
  3. Spectral Analysis: NDVI (vegetation), NDBI (built-up), NBI (new construction) index computation
  4. ML Classification: Random Forest trained on 6 spectral bands, validated with 70/30 split
  5. Change Detection: Bitemporal differencing of spectral indices with adaptive thresholds
  6. Violation Extraction: Connected component analysis with geo-coordinate conversion
